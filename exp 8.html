Quick reference

/* key flexbox properties used above */
.container { display: flex; /* makes children layout with flexbox */ }
.container { flex-direction: row | column; /* main axis: row (default) or column */ }
.container { justify-content: flex-start | center | space-between | space-around; /* alignment along main axis */ }
.container { align-items: flex-start | center | stretch; /* alignment along cross axis */ }

Tip: Use flex: 1 on the element you want to grow and fill available space. Use media queries to switch flex-direction on small screens.
